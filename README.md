## **CourtSync: Padel Court Reservation System**

**CourtSync** is an automated, high-performance digital solution designed to bridge the gap between Padel facility providers and players. Developed as a robust **Java-based** application, the system replaces inefficient manual coordination with a synchronized, real-time booking ecosystem that ensures accuracy and ease of use.

The platform is engineered with a clean separation of concerns, utilizing an **MVC (Model-View-Controller)** architecture and optimized data persistence to handle the high-concurrency demands of a reservation marketplace.

---

### **🚀 Core System Features**

* **Intelligent Search & Discovery:** A comprehensive module that allows players to find venues based on location, city, or proximity, ensuring athletes can find a match wherever they are.
* **Real-Time Availability Tracking:** An integrated logic engine that cross-references user queries with the database to display only live, bookable time slots.
* **Synchronized Booking Engine:** A Java-powered backend that manages the entire reservation lifecycle—from slot selection to final confirmation—while preventing double-bookings through atomic state management.
* **Secure User Management:** A dedicated identity layer providing encrypted registration and role-based access for both Customers and Administrators.
* **Administrative Suite:** A management interface for facility owners to update court metadata, manage schedules, and view usage trends to optimize venue operations.

---

### **🛠️ Technical Architecture**

* **Core Language:** Java (JRE 11+).
* **UI Framework:** Java Swing, utilizing custom UI styles for a modern, uniform user experience.
* **Data Management:** Efficient, persistent storage (CSV-based for lightweight evaluation) with atomic file replacement to ensure data integrity.
* **Security:** Implementation of secure credential hashing and strict input validation to prevent injection and unauthorized access.
